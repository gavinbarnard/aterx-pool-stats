Get started

  • Offical wallet software
  • Download Xmrig or build Xmrig from source
  • Add to Windows Defender
    1. Windows Defender -> Exclusions -> Virus & Threat protection -> Virus & threat protection settings -> Settings -> Exclusions
    2. Add xmrig.exe as a process name, and the folder where it is located otherwise Windows Defender will delete it.
  • Edit config file or run via comand line
  • Example Pools section of config.json: 
    					"pools": [
						{
							"algo": "rx/0",
							"coin": null,
							"url": "pool.aterx.com:4244",
							"user": "<wallet>",
							"pass": "d=27881",
							"rig-id": "<custom-rig-id>",
							"keepalive": true,
							"enabled": true,
							"tls": true,
							"tls-fingerprint": null,
							"daemon": false,
							"socks5": null,
							"self-select": null
						}
					]
  • Note: If the pools section is not the last item then there should be a , after the ] of the pool section
  • Run Xmrig by right click Xmrig -> Run as administrator or run Xmrig from the CLI 
    xmrig -o pool.aterx.com:4244 --tls --keepalive -u <walletid> -p d=20000 --algo rx/0 --rig-id <CustomRigIdentifier>
  • On initial launch it may say to enabled huge page but that a reboot is needed, it is highly recommended to reboot to get a better hashrate
  • Recommended starting d=value is 20 * hashrate (defualt is 1000 and be auto adjusted up).
